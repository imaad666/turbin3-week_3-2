# NTRS Vault

A Solana vault program built with Anchor framework.

## Instructions

- **Initialize** - Create a new vault for the user
- **Deposit** - Deposit SOL into the vault
- **Withdraw** - Withdraw SOL from the vault
- **Close** - Close the vault and recover all lamports
